¿Qué es esta plataforma?

BienestarApp es una plataforma para ayudarte a llevar un control organizado de tus hábitos saludables. Puedes registrar desde caminatas y meditación hasta lectura o yoga.

Es rápida, segura, intuitiva y completamente construida como SPA con JavaScript vanilla, CSS limpio y una API backend real.
